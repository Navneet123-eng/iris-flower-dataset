🌸 Iris Flower Dataset Project
📌 Overview
This project uses the famous Iris flower dataset to perform data analysis, visualization, and machine learning. The goal is to understand patterns in flower measurements and classify species accurately.
📊 Dataset Description
The dataset contains 150 samples of iris flowers with the following features:
Sepal Length
Sepal Width
Petal Length
Petal Width
Species (Target Variable)
There are three species:
Setosa
Versicolor
Virginica
⚙️ Project Tasks
Data Exploration (EDA)
Data Visualization
Model Training using Machine Learning
Classification of Iris Species
🤖 Machine Learning Model
We use Linear Regression / Classification models to predict the species based on input features.
